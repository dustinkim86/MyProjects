## 예측 결론

- 브랜드 별 매장 수의 차이는 크지만 **인지도**에는 크게 **차이가 없고**, **가까울수록** 이용하는 경우가 많기 때문에 가장 최적의 입지 위치를 분석하여 올리브영에 대응하는 솔루션을 제안할 수 있을 것으로 판단됨
